Cómo usar “Permisos del Departamento”

¿Para qué sirve? Esta pantalla permite definir permisos propios sobre una carpeta (departamento) para usuarios o grupos concretos.

  1. Herencia de permisos: activada hereda del sitio padre; desactivada usa permisos propios de la carpeta.
  2. Permisos heredados: son solo informativos y no se editan aquí.
  3. Permisos propios: puedes añadir, cambiar rol o eliminar permisos locales.
  4. Agregar permiso: busca un grupo/usuario, elige rol y acceso, luego pulsa “Agregar”.
  5. Guardar cambios: confirma todos los ajustes realizados.

Nota: Solo disponible para administradores globales.
